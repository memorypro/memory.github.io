    树莓派安装php和apache

    发布于 2019-05-18 | 分类于 技术 | 1分钟 | 186字数

    更新源

    sudo apt-get update

    安装apache2

    sudo apt-get install apache2

    安装过程退提示你输入 Y

    浏览输入树莓派的地址出现apache页面即为安装成功，放置网页的路径在 “/var/www/html”

    安装php

    sudo apt-get install php5

    安装Sqlite

    sudo apt-get install sqlite sqlite3

    安装sqlite 的php组件

    sudo apt-get install php7.0-sqlite

    常用命令

    sudo /etc/init.d/apache2 start   // 启动服务器
sudo /etc/init.d/apache2 stop    // 停止服务器
sudo /etc/init.d/apache2 restart // 重启服务器

    踩过的坑

    1、安装sqlite 的php组件

    正确:

    sudo apt-get install php7.0-sqlite

    错误：

    sudo apt-get install php7-sqlite

    2、 vi编辑器 上下左右出现 ABCD

    sudo apt-get install vim-gtk

    3、出现 Call to undefined function utf8_decode()

    sudo apt install php-xml
