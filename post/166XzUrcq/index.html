Gridea 主题 Terrible

发布于2020-01-08| 3分钟| 555字数

概述

程序员造轮子，之前修改了别人的一款主题NexT，但是有一些问题没有得到妥善的解决，比如响应式布局等等，所以自己重新从头开始制作了一款主题Terrible。

Terrible:可怕的，整个主题完整的黑白色，包括头图的选择都是黑白色

主题功能

  1. 头图提供两种颜色的样式供选择。
  2. 支持Prism代码高亮
  3. Vline 评论插件
  4. 根据标签文章分类
  5. 带搜索框但是不支持搜索
  6. 支持谷歌统计和百度统计
  7. 添加百度主动推送
  8. 可以添加友情链接
  9. 可以自己添加社交信息
  10. 引入bootstrap，支持响应式布局。
  11. 每篇文章随机显示头图，头图目前不能自定义，后期会支持自定义内容。

安装主题

1、下载主题

git clone https://github.com/memorypro/gridea-theme-Terrible.git

2、下载完成解压后将文件夹拷贝到"Gridea\themes"路径下面。

3、重启Gridea 在主题菜单栏选择Terrible.

4、使用prismjs支持代码高亮，

​ 注意：一定要使用prismjs，因为我把主题默认的代码高亮highlight删掉了😘。

​ 打开 https://prismjs.com/ 根据自己喜欢的风格下载prism.js文件和prism.css文件，将这两个文件放到Gridea/static/路径下面，重新生成即可。

5、自定义配置

  • 外观配置: 打开Gridea客户端在自定义配置页面，外观栏目可选择两种Header风格。

  • 评论系统:

    • 访问https://leancloud.cn/，如果有Github账号，直接登陆，然后需要绑定下邮箱即可。
    • 进去之后创建应用，开发版即可。
    • 点击右上角进入到设置界面。点击应用Key，复制里面的AppID 和AppKey到Gridea客户端下面的AppID和AppKey里面即可。

  • 百度统计和谷歌统计，需要注册百度账号和谷歌账号，复制对应的谷歌跟踪ID和百度追踪代码到Gridea即可。

  • 友情链接和社交信息目前只支持网址和名称，图标设置无效。
