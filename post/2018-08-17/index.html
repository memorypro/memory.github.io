Ubuntu 安装java 1.8 教程

发布于2018-08-17 | 2分钟| 391字数
1、下载java 1.8

地址：

​ http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

注意：

​ Ubuntu :.tar.gz

​ Redhat:.rmp

2、创建java 文件夹

在根目录下面创建java文件夹


$ cd /
$ sudo mkdir java
$ cd java

注意：java 文件夹可以自由放置，不一定要放在根目录下面。

3、上传java安装包

将java 安装包放到刚才新建的java目录下面，如果在windows下面可以使用Winscp 上传。

WinScp 使用：

​ http://www.good-memory.top/2018/4/27/blog.html

4、解压安装包

使用下面的命令解压安装包


$ tar zxvf FileName.tar.gz
5、配置环境变量

配置环境变量，我们需要知道JDK的路径，使用下面的命令

;打开java路径
memory@ubuntu:~$ cd java
memory@ubuntu:~/java$ ls
jdk1.8.0_11  jdk-8u11-linux-i586.tar.gz
;打开JDK路径
memory@ubuntu:~/java$ cd jdk1.8.0_11/
;浏览当前路径的绝对路径
memory@ubuntu:~/java/jdk1.8.0_11$ pwd
/home/memory/java/jdk1.8.0_11
memory@ubuntu:~/java/jdk1.8.0_11$

如上所示，我们拿到绝对路径

打开/etc/profile文件配置环境变量

vi /etc/profile

写入如下内容

#Java Env
export JAVA_HOME=/home/memory/java/jdk1.8.0_11   
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
export PATH=$PATH:$JAVA_HOME/bin

注意第二行后面的值是java的绝对路径。

6、检查java是否安装成功

使用下面命令检查java是否安装成功

$ java -veision
java version "1.8.0_11"
Java(TM) SE Runtime Environment (build 1.8.0_11-b12)
Java HotSpot(TM) Client VM (build 25.11-b03, mixed mode)
memory@ubuntu:~/java/jdk1.8.0_11$

出现上面的版本好便是安装成功了。
