GIT 提交代码到远程仓库

发布于2019-12-27 | 2分钟| 253字数

概述

Git是什么？
Git是目前世界上最先进的分布式版本控制系统（没有之一）。
Git有什么特点？简单来说就是：高端大气上档次！

操作流程

1、初始化本地仓库

git init

2、克隆远程仓库代码

git clone https://gitee.com/zyzp/QT-SerialPort.git

3、添加代码到仓库

cd //
git add .

4、提交代码

$ git commit -m "提交代码"
